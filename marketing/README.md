## Order of operations

1. Ship Parts A–D (technical SEO, on-page, compare pages, blog) — **done** in this repo.
2. Polish the **main repo README** ([template](agentflow-readme-template.md)).
3. Verify the docs site is indexed (Search Console → URL inspection on 5–10 key pages).
4. Submit to **awesome-lists** ([`awesome-list-submissions.md`](awesome-list-submissions.md)) — backlinks.
5. **Show HN** + Reddit posts the same week. Use HN as the anchor; Reddit as amplification.
6. **Dev.to / Hashnode cross-posts** of cornerstone blog articles, with `canonical` set to the docs site.
7. **Outreach to listicle authors** ranking for "best agent framework Python" — give them a useful angle, not a spam pitch.
8. Track everything in [`measurement-playbook.md`](measurement-playbook.md). Review weekly.

## A note on tone

Every asset here is written to be honest. We do not trash competitors. We position AgentFlow's strengths and acknowledge when other frameworks are the right pick. This keeps the brand defensible — and Google's helpful-content updates penalize the alternative.

If you find yourself writing "Other frameworks are slow / bad / broken," delete it.
