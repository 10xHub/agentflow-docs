# Tutorials

:::note Draft
This page is a sprint-0 placeholder for the tutorial catalog. Tutorial pages should not be treated as complete until their code has been validated.
:::

Tutorials will be written as complete learning tracks, not isolated snippets.

## Planned beginner track

1. Build your first agent.
2. Add a Python tool.
3. Add state and memory.
4. Build a two-agent handoff.
5. Add checkpointing.
6. Expose the workflow through the API.
7. Stream responses to the TypeScript client.

## Planned production track

1. Structure an AgentFlow project.
2. Add persistent storage.
3. Add background tasks and callbacks.
4. Add evaluation.
5. Deploy the API.
6. Connect the frontend.

Each tutorial should end with a working state and a short explanation of what the reader can safely change next.
