# Beginner path

:::note Draft
This page is a sprint-0 placeholder. The full beginner path will be written and validated in sprint 1.
:::

The beginner path will teach AgentFlow one concept at a time. It starts with a single Python agent, then adds tools, state, memory, the API runtime, the hosted playground opened by `agentflow play`, and a TypeScript client call.

## Planned path

1. Mental model.
2. Your first agent.
3. Add a tool.
4. Add memory.
5. Run with the API.
6. Test with the hosted playground.
7. Call from TypeScript.

## Writing rules

Every page in this path should include expected output, a short "What you learned" section, and one recommended next step.
